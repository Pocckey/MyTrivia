Welcome to my trivia game

Textbook: Game Programming All In One (third edition) by Jonathan S. Harbour
---------------------------------------------------------------------------------------------
Run file in Release/Trivia.exe
if that does not work, run trivia.sln

---------------------------------------------------------------------------------------------
The trivia game uses all the questions from the textbook

Buttons used: UP, DOWN, LEFT, RIGHT, ENTER, ESC
---------------------------------------------------------------------------------------------
To add or change a questions: edit the Qsts.txt file

To add or change an answer: edit the Ans.txt file
---------------------------------------------------------------------------------------------
To change the music: add the music to the project file, and change the link in the main.cpp
Default music + Sound effect: 
http://www.soundjig.com/pages/music/jazz.html 
- Smooth latin congos with chello.wav

http://www.soundjay.com/button-sounds-3.html
-Button Sound 30.wav
---------------------------------------------------------------------------------------------

Game volume can be changed in game at the very beginning. Just press up or down
---------------------------------------------------------------------------------------------
After changing volume level, users will be prompt to select chapter
Just press up or down then hit enter to go to the chapter you want to go to
---------------------------------------------------------------------------------------------
To change font color to red or white: press Left, Right at any time then press Enter
---------------------------------------------------------------------------------------------

When the quiz is complete, the program will automatically exit
---------------------------------------------------------------------------------------------
